Testing Checklist:

CLI:
Usage: ccmsh [options]

Options:
  -h, --help       show this help message and exit

  List existing emulations:
    -l, --list     [emulationName]  List of emulations by ID or all
    -r, --results  [emulationName] [all]  List of emulation results
    --list-jobs    [all] List of all active Jobs running on Scheduler daemon

  List available distributions:
    -i, --dist     [all] or [name] List of all available distributions

  List available emulators:
    -e, --emu      [all] or [name]  List of all available emulators

  Create new emulations:
    -x, --xml      [file.xml] enter name XML document
    -n, --now      [-x filename] plus [-n] emulation starts immediately

  Delete emulations:
    -d, --delete   [emulationID] Deletes full emulation
    --purge        [all] clears everything in DB. !!USE WITH CAUTION!!

  Services Control:
    --start        [scheduler][api] followed by interface and port to start
                   services. Example: ccmsh --start api eth1 2020
    --stop         [scheduler][api] stop services
    --show         [scheduler][api] show services
    -v, --version  Display version info
=================================++++==============================================

5. List available distributions/emulators:

6. Delete emulations:
